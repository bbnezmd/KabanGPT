# KabanGPT
Local AI assistaint using DeepSeek architecture

This is my mini-project called KabanGPT. It was written on a HTML + CSS + JS. The full development lasted about 3 days 

To use the KabanGPT, you need a PC (Windows, Mac, Linux, whatever...), Internet connection and Ollama

Download Ollama:
- Windows — [ollama.com](ollama.com)
- Linux — ```sudo apt install ollama```

Then turn Ollama on and download the version of KabanGPT. The newest is 1.1.2 \
On linux — ```ollama serve```

After that you can use it as much as you wish. It is absolutely free and open-sourced!
